And please...

Don't forget the media query `max-width`

This code smells

div { display: none }

@media all and (min-width: 768px) {
  div { display: block }
}

You can write this

@media all and (max-width: 767px) {
  div { display: none }
}

or this

@media not all and (min-width: 768px) {
  div { display: none }
}
